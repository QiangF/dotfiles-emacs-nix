* Util functions

A slight hack to add around functions as advice to prevent cursor from jumping around with some actions

#+BEGIN_SRC emacs-lisp
(defun save-excursion-wrapper (inner &rest args)
  (save-excursion
    (funcall inner)))
#+END_SRC
